# fun_code

1. swap_string_inquote.py

    This code will return the string with swapped characters inside the quotes.    
    Example input/output expectation 
    
        * abc(de(fg)hi) => abcihfged        
        * a(bcdefghijkl(mno)p)q => apmnolkjihgfedcbq        
        * abc(de)f => abcedf        
        * abcdef => abcdef
